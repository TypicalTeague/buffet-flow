# BuffetFlow

BuffetFlow is a machine learning-powered system designed to reduce food waste in buffet-style dining environments such as university dining halls.

## Problem
Buffet dining creates significant food waste because staff must decide how much food to prepare and refill while customers serve themselves unpredictably. This leads to overproduction, inefficient refills, and high levels of both kitchen and plate waste.

## Solution
BuffetFlow predicts item-level demand and waste, then provides real-time refill recommendations such as full refill, half refill, or stop refill. This helps dining staff make better decisions during service, reducing waste without risking food shortages.

## Features
- Predicts buffet item demand
- Estimates likely waste per item
- Provides refill recommendations
- Simulates real buffet dining scenarios
